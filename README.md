# Emeka Amadi Foundation - Internal Documentation

This repository contains the source code for the Emeka Amadi Foundation website UI, intended for internal use by company employees for UI maintenance and updates.

## Cloning and Rebuilding

To clone and rebuild the Emeka Amadi Foundation website UI for internal use, follow these steps:

1. **Clone the Repository:**

   ```bash
   git clone git@github.com:jkavod/eaf-ui.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd eaf-ui
   ```

3. **Install Dependencies:**

   ```bash
   yarn install
   ```

4. **Start the Development Server:**

   ```bash
   yarn start
   ```

   This command will launch the development server and open the webpage in your default web browser at `http://localhost:3000`.
